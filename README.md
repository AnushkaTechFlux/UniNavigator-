# UniNavigator
Premium glassmorphic college prediction dashboard built with PHP &amp; MySQL. Instantly match JEE, BITSAT, &amp; Board scores to premier campuses.
UniNavigator is an interactive, high-fidelity college prediction and academic routing system designed to help engineering aspirants find their ideal university matches. Rebuilt from a classic educational predictor structure, it features a premium, responsive dark-glassmorphic environment inspired by modern SaaS applications.

By analyzing high school Board exam grades alongside scorecards from key competitive engineering entrance examinations (JEE Main, BITSAT, VITEEE, and SRMJEEE), UniNavigator computes a weighted average index and instantly recommends matches across premier Indian institutions (IITs, NITs, BITS, VIT, SRM).

✨ Core Features
💻 Premium Dark-Glassmorphic UI: Beautifully designed transparent elements using pure CSS backdrop-filter: blur(20px), subtle border-glow strokes, and glowing radial mesh gradients.
🔐 Secure Workspace: Built-in PHP session verification securing the dashboard page, preventing unauthorized guest access.
📊 Smart Score Calculations: An optimized mathematical evaluator that checks board thresholds (60% min) and balances grade weights with entrance exam results.
🎓 Dynamic Campus Match Cards: Eligibility outputs are displayed inside modern, color-coded campus cards complete with tier badges and score criteria badges.
⚙️ Relative Routing (Zero-Config): 100% removal of hardcoded local folders; the application runs out-of-the-box on XAMPP/Apache under any subdirectory.
🔄 Smooth Transition Animations: Micro-animations and jQuery-powered fade transitions for a seamless login and registration switch.
